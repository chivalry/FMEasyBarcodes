FMEasyBarcodes Setup and Configuration: Parameters

Here is information about the parameters / settings that are supported by FMEasyBarcodes...

VALUE
The value to be displayed on the barcode.

SYMBOLOGY
The type of barcode (symbology) to be generated. Supported options include:

Symbology Option
EAN 8 ean8
EAN 13 ean13
UPC upc
Standard 2 of 5 std25
Interleaved 2 of 5 int25
Code 11 code11
Code 39 code39
Code 93 code93
Code 128 code128
Codabar codabar
MSI msi
Data Matrix datamatrix
QR Code qrcode

WIDTH
For barcodes, this is the width of the bars generated. For QR codes, this is the width of the QR code that will be generated.

HEIGHT
For barcodes, this is the height of the bars generated. For QR codes, this is the height of the QR code that will be generated.

FGCOLOR
The color of the barcode or QR code generated.

BGCOLOR
The background color of the barcode.

FMPURL
An optional FMP URL to use to send the generated barcode back to FileMaker. For example, the FMP URL used in the the demo database is: fmp://$/FMEasyBarcodes.fmp12?script=FMP URL Handler - Save Barcode to Container

In that example, the "FMP URL Handler - Save Barcode to Container" script is what gets run after the barcode has been generated. It takes the Base64-encoded barcode, decodes it, and stores it in the database.
