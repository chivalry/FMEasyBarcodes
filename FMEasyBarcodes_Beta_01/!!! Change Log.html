FMEasyBarcodes Change Log

Beta 1 (10-31-2014)
• Initial public beta.

Beta RC2 "Smasher" (10-30-2014)
• Merged code to support barcodes and QR code in the same template.
• Completed demo FileMaker database.
• Added support for additional settings, including CSS for DIV.

Beta RC1 "Sizzler" (10-29-2014)
• Fully consolidated file, including jQuery, Sizzle, BarCode Coder Library, jquery.qrcode.js, and HTML.
• Token substitutions are now in place.

Alpha 2 (10-03-2014)
• Second alpha release to small test group.
• Initial build w/ externally referenced HTML, jQuery, JS libraries, etc.
• Limited support for QR codes. Code is fragmented but working.
• No support for Windows.

Alpha 1 (10-01-2014)
• Alpha release to clients only.
• Supports all major barcode types.
• No support for QR Codes.
• Several hard-coded values still need to be configured as settings.
